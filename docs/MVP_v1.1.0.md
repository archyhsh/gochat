# GoChat MVP v1.1.0

## 🎯 项目目标
实现高性能分布式 IM 核心，具备完整的社交和聊天基础体验。

## 🏗️ 核心架构
- **Gateway**: WebSocket 连接管理与消息实时转发。
- **User**: 账户体系、JWT 鉴权、个人资料管理。
- **Relation**: 好友关系、群组管理、实时黑名单。
- **Message**: 消息持久化、历史查询、未读数统计。
- **Infra**: Kafka (异步解耦)、MySQL (分月存储)。

## ✅ 已实现功能

### 1. 社交管理
- [x] **用户**: 注册登录、Session 持久化、资料修改。
- [x] **好友**: 搜索添加、申请处理、备注修改、删除。
- [x] **黑名单**: 实时拉黑/取消，拦截消息推送。

### 2. 聊天系统
- [x] **即时通讯**: 1v1 私聊、多人群聊（WebSocket）。
- [x] **可靠性**: 消息通过 Kafka 异步写入 MySQL。
- [x] **交互体验**: 
    - 滚动锚定（自动置底/加载更多）。
    - 实时红点（未读数统计）。
    - 会话置顶（新消息自动上移）。
    - 正在输入提示（私聊场景）。

### 3. 群组增强
- [x] **群管理**: 创建群（支持初始邀请）、搜索发现、申请入群。
- [x] **成员管理**: 成员列表、邀请、退出/解散。

## 🚀 快速启动
1. **环境**: `docker-compose up -d mysql kafka`
2. **初始化**: 执行 `deployments/docker/mysql/init.sql`。
3. **编译**: `go build -o bin/ ./cmd/...`
4. **运行**: 启动 `bin/` 下的四个服务。
5. **访问**: `http://localhost:8080`
